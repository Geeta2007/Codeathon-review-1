# Codeathon-review-1
This project is a responsive and user-friendly web application developed using modern web technologies. The website is designed to deliver a seamless user experience with a clean layout, intuitive navigation, and structured content presentation.

#Features
Clear and structured university information
Dedicated placement section with guidance to help students prepare for and crack placements
Analysis of placement data to provide insights and trends
User-friendly navigation and responsive design

#Tech Stack
HTML5
CSS3
JavaScript

#Usage
Clone the repository and open index.html in a browser to explore the website.

#Purpose
The goal of this project is to provide an intuitive platform for users to explore university details while also supporting students with placement preparation and data-driven insights.
